Welcome to the Symbaroum Core Rules module.


If you are new to the Symbaroum RPG system make sure you check out the @JournalEntry[Symbaroum System guide EN] Journal which can be found in Compendiums>Symbaroum for FVTT system user guides.

All you need to run a game is imported and organised into folders for each of the asset types.
You can rerun the import script at any time by going to Configure Settings> Module Settings and hitting the import button. Any existing assets will not be overwritten so it's useful if you accidentally delete something.

NOTE:
Future updates will alter the data in the folders described below but will not touch any folders or content you create outside of them.
To prevent losing any of your active scenario/world data is is advisable to duplicate any characters or creatures you are using and put them in your own folders. This will ensure they are not overwritten by the upgrade script.


Journals

Under the "Symbaroum Core Rules - Journals" folder you will find the following Journals which contain the material for each of the main sections.

  • Book 1: The World of Symbaroum Entries containing descriptions of the world as well as key locations. (GMs should give the players Observer permissions to this folder)
  • Book 2: The Players Guide Everything the players need to know about creating characters, equipment and rules. (GMs should give the players Observer permissions to this folder). This also includes the "Alternative Rules" chapter from the Advanced Players Guide.
  • Book 3: The GM Guide The game rules, campaign rules, adversaries and monsters.
  • Book 4: A Terra Prometida The three part introductory scenario for GMs and players.
  • 1. The Art of Symbaroum Images from the book organised into three sections to use during your adventures.

Under the "1. The Art of Symbaroum" folder you will find Journal entries with Images from the book organised into three sections to use during your adventures:

  • Book 1
  • Book 2
  • Book 3

Actors

The "Symbaroum Core Rules - Actors" folder has subfolders for Adversaries, Monsters, Mystical Power Familiars containing the people and creatures the PCs might meet during their journeys through Davokar.

In the "1. The Promised Land - Actors" folder you will find the PCs and NPCs for The Promised Lands scenario which are set up as per the scenario with any necessary equipment, abilities, traits, etc.

"Symbaroum - APG - Actors" contains templates for the creatures for the Thorn Beasties Elixir, and the Death Lord, Rune Guardian, Servant Daemon and Summon Daemon Rituals.


Scenes

Core Rules - Book 1 contains all the maps relevant to the The World of Symbaroum journal entries.

The Promised Land - Scenes has the maps as set out in the book for the scenario.

GM Aids has two maps of Ambria and Davokar, one with coloured hexes and one without, as well as the Rules Cover and the Quick Guide for Combat.

The Quick Guide for Combat is presented as a scene as it is the clearest way to present the information.
It is set to show on the navigation bar to all players allowing them to switch to it when they need to reference any information.

Map pins have been added where applicable. Please note that some have blank journal entries because there are no descriptions in the book.

As Symbaroum is mainly a "Theatre of the Mind" system, no walls have been added to the maps.


Items

Items are organised into two folders:

  • Symbaroum - Items - Contains all the assets from the Core Rules Book.
  • Symbaroum - APG - Items - Has all the assets from the Advanced Players Guide.

Under each of these folders the items types are organised into individual folders as follows:

  • Abilities*
  • Alchemical Elixirs
  • Armor*
  • Artifacts
  • Boons/Burdens*
  • Equipment*
  • Monstrous Traits
  • Powers*
  • Rituals*
  • Traits*
  • Weapons*

As the GM you may wish to give the players Observer permissions to some of the folders (suggested folders are marked above with an * ). The Weapons and Armor folders contains the generic templates for weapon and Armor types (Heavy Armor, Light Armor, Heavy Weapon, Long Weapon etc.).
By duplicating these templates the GM can craft any required weapon (i.e. Use the Long Weapon to create a Spear, Light Armor to create Studded Leather).


IMPORTANT NOTE REGARDING BOONS/BURDENS:
In the Symbaroum Advanced Player Guide a number of Traits were reclassified as Boons/Burdens.
As the Advanced Players Guide Boons/Burdens are now included in the core rules module, Bushcraft, Contacts, Long-Lived, Pariah, Privileged and Short-Lived, that were included in the Core Rules now have XP costs associated with then as per the APG where a Boon is bought for 5xp/level and a Burden is -5xp/level. This is fully explained in the Advanced Players Guide page 50.


Compendiums

The Symbaroum modules now use the Foundry Adventure Import system, so there is only a single Compendium which contains all of the assets for the module.

The import should have created folders for everything you need to run Symbaroum.

You can rerun the import at any time by going to "Configure Settings > Module Settings" and hitting the Re-import button. Re running the import will only replace missing content and will not overwrite anything.


Recommended Add-on Modules

While not necessary to use the Symbaroum system or this module I would recommend installing the following Add-on modules as they make the GM's job a lot easier:

  • Dice So Nice:
    Because it's awesome!
  • Token Action HUD:
    Great tool for allowing rolls by just selecting the token and clicking on the HUD.
  • PnP - Pointer and Pings:
    Useful for pointing things out to the players and pulling their map focus to where you want it.
  • Quick Insert - Search Widget:
    Allows you to quickly search for assets and drag them directly on to the scene.
